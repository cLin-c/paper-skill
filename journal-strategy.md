# Journal Strategy & Research Question Module

> **Core philosophy (from Nature-Paper-Skills):** Select your target journal BEFORE writing, not as a last resort after rejection. Journal selection shapes argument depth, figure standards, word limits, and framing — writing first and targeting later wastes weeks of revision.

## 1. 期刊优先决策

**在写作开始前**，先确定目标期刊。期刊决定了文章的定位、深度和格式。

### 期刊选择（写作前必做）
```
我的研究方向和核心贡献：
- 领域：[领域]
- 核心方法/发现：[一句话描述]
- 数据规模/实验规模：[规模]
- 主要对比基线：[基线方法]
- 预期受众：[工程师/科学家/临床医生/跨学科]

请推荐3-5个最适合的目标期刊，对每个期刊给出：
1. 期刊名称 + 出版商 + ISSN
2. Impact Factor（近3年均值）和分区（JCR Q1/Q2/CAS）
3. 该期刊的核心 Scope（哪类工作最易发）
4. 近期（2023-2026）发表的与我研究最相关的论文标题（2-3篇）
5. 特别要求（页数限制/图数限制/数据共享要求）
6. 审稿周期（均值）和录用率（若已知）
7. 是否支持中文作者（是否有语言编辑服务）

目标领域方向：[SCI/EI/Nature系/IEEE/ACM/IJCAI/AAAI/NeurIPS/CVPR/ICCV/医学类/其他]
```

### 快速期刊匹配（5分钟决策）
```
我有一篇[方法类/应用类/综述类/临床类]论文，核心亮点是：
[一句话描述核心贡献]

快速给出：
- 最高目标（可能拒稿但值得一试）：
- 主要目标（最匹配的1-2个期刊）：
- 保底选择（高录用率的可靠期刊）：

每个期刊说明：为什么适合我的研究？需要如何调整论文侧重？
```

## 2. FINER研究问题评分

FINER 框架（Feasible / Interesting / Novel / Ethical / Relevant）用于在写作前评估研究问题质量。

### FINER全面评分
```
请对我的研究问题进行FINER评分：

研究问题：[明确描述你的研究问题，1-2句话]

按以下5个维度各打1-5分，并给出详细理由：

F — Feasible（可行性）1-5分
  评估要点：
  - 在现有资源/数据/计算条件下能否完成？
  - 时间范围内能否得出有意义的结论？
  - 团队是否具备必要的专业知识？

I — Interesting（重要性）1-5分
  评估要点：
  - 该问题对目标期刊的读者群是否重要？
  - 解决该问题是否会推动领域进展？
  - 是否有研究社区会关注这个结果？

N — Novel（新颖性）1-5分
  评估要点：
  - 与现有发表的工作有何本质区别？
  - 是否只是已有方法的微小变体？
  - 新颖点是方法、数据、发现还是应用场景？

E — Ethical（伦理性）1-5分
  评估要点：
  - 是否需要伦理审批（人体/动物实验）？
  - 数据来源是否合法合规？
  - 研究结果的潜在社会影响是否可接受？

R — Relevant（相关性）1-5分
  评估要点：
  - 是否与目标期刊的Scope高度匹配？
  - 是否与当前领域热点/挑战直接相关？
  - 能否与政策/临床/工程实践挂钩？

总分 = F + I + N + E + R（满分25）

解读：
≥ 22：研究问题优秀，可直接进入写作阶段
18-21：良好，建议强化得分最低的1-2个维度
14-17：需要修改研究问题，重点改进低分维度
< 14：建议重新定义研究问题

给出针对低分维度的具体改进建议，以及改进后的研究问题表述
```

### 研究问题精炼
```
我的初始研究问题是：
"[初始研究问题]"

FINER评分显示[N=2/R=3]等维度较弱。

请帮我精炼这个研究问题：
1. 新颖性强化：现有工作解决了A和B，我的工作聚焦于尚未解决的C——如何在问题表述中突出C？
2. 相关性强化：如何将研究问题与[目标期刊]的核心读者关切更紧密地联系？
3. 范围调整：问题是太宽泛（无法在一篇文章中完整回答）还是太窄（贡献不足以发顶刊）？

给出3个改进版本的研究问题，并对每个版本给出FINER预评分
```

## 3. 期刊风格校准

不同期刊有独特的写作风格、论证密度和格式偏好。写作前校准可显著提高录用率。

### 风格校准（分析样本论文）
```
我计划投稿到：[目标期刊名]

我提供了该期刊近期发表的[5-10]篇相关论文（或提供标题列表请你分析）。

请帮我生成这个期刊的"风格校准卡"，分析以下维度：

1. 摘要结构
   - 典型句数和词数？
   - 是否有固定段落结构（背景/方法/结果/意义）？
   - 第一句通常如何开始（问题/发现/挑战）？

2. 引言写法
   - "漏斗"结构还是"问题-解决"结构？
   - Gap通常在第几段出现？
   - 本文贡献通常以什么格式列出（bullet points / paragraph / numbered）？

3. 结果呈现
   - 是先陈述发现再给数据，还是数据嵌入陈述？
   - 图表caption通常多少词？是否包含统计结果？

4. 讨论/结论
   - 是否独立分节还是合并 Discussion & Conclusion？
   - 局限性通常在哪里讨论？

5. 语言风格
   - 平均句长（短句/长句风格）？
   - 主动语态vs被动语态比例？
   - 是否使用第一人称（We/I）？

6. 引用密度
   - 引言每段平均几条引用？
   - 方法部分引用模式？

输出：一页"[期刊名]风格校准卡"，我可以在写作时随时参考
```

### 单节风格对齐检查
```
请将我的[摘要/引言/方法/结论]与[目标期刊]的风格要求对比：

我的当前写法：
[粘贴你的段落]

[目标期刊]的典型风格特征：
[粘贴风格校准卡的相关条目，或让AI根据期刊名自行分析]

请指出：
1. 哪些地方与该期刊风格不符？
2. 具体修改建议（不是重写整段，而是精准的改动）
3. 修改后的版本

重点关注：句式结构、逻辑顺序、是否符合期刊的论证风格
```

## 4. 研究空白系统识别

投稿顶刊的论文需要清晰定位研究空白，且每个gap必须有充分文献支撑。

### 系统性研究空白分析
```
我正在写一篇关于[主题]的论文。

请帮我系统性分析研究空白：

步骤1 — 现有方法/知识地图：
列出该领域最重要的5-8个现有方法/研究，按类别分组：
- 方法A类：[列表]（核心思路：xxx）
- 方法B类：[列表]（核心思路：xxx）

步骤2 — 每类方法的局限性识别：
对每类方法，明确指出：
- 技术局限（做不到什么）
- 应用局限（在哪些场景失效）
- 数据局限（需要什么条件才能用）

步骤3 — Gap定位（我的论文填补哪个）：
我的方法：[描述]
我填补的gap是：[现有方法X的局限Y，我的方法Z解决了它，证据是W]

步骤4 — 引用支撑建议：
每个gap声明需要3-5条直接引用（不是背景性引用）：
- [Gap 1]：建议搜索关键词：[xxx]，推荐引用方向：[xxx]
- [Gap 2]：建议搜索关键词：[xxx]，推荐引用方向：[xxx]

输出：完整的研究空白分析，可直接用于写作引言
```

### Gap声明质量检查
```
我引言中的研究空白声明如下：

[粘贴你的gap段落]

请检查每个gap声明的质量：
1. 声明是否足够具体？（"现有方法精度低"是模糊的，"现有方法在遮挡场景下精度下降超过30%"是具体的）
2. 每个gap是否有至少2-3条直接支撑引用？（而非仅主题相关引用）
3. Gap是否真实（有文献支撑）还是基于个人推断？
4. 从Gap到我的解决方案的逻辑过渡是否流畅？

逐条评估，给出强化建议和修改后的Gap段落
```

## 5. 梯队投稿策略

### 三梯队投稿规划
```
我有一篇[论文类型]论文，研究方向：[方向]

请帮我制定三梯队投稿策略：

梯队A（冲刺）：
- 目标：[期刊名]（IF/等级：xxx）
- 为何适合：[理由]
- 需要加强的地方：[具体调整]
- 预计审稿时间：[xxx]

梯队B（主要目标）：
- 目标：[期刊名]（IF/等级：xxx）
- 为何适合：[理由]
- 与梯队A相比需要如何调整：[调整]
- 预计审稿时间：[xxx]

梯队C（保底）：
- 目标：[期刊名]（IF/等级：xxx）
- 为何适合：[理由]
- 预计审稿时间：[xxx]

注意：同一时间只能投一个期刊（不得同时投稿）。给出各梯队投稿顺序建议和每次被拒后的修改重点。
```

### 转投期刊时的论文调整
```
我的论文被[原期刊]拒稿，审稿意见是：
[粘贴主要拒稿理由]

我计划转投[新期刊]。

请帮我分析转投需要做哪些调整：
1. 两个期刊的Scope差异是什么？我的论文侧重点需要如何转移？
2. 针对拒稿意见，哪些是我必须修复的（新期刊也会提出），哪些是期刊特异性的（可以不改）？
3. 格式调整清单（字数/图数/引用格式/摘要结构）
4. 哪个部分需要重写，哪个部分可以直接复用？

生成转投调整计划和优先级排序
```

## 6. 投前适配检查

### 投稿前期刊适配核查
```
我准备将这篇论文投稿到[期刊名]，请进行投前适配检查：

内容适配：
□ 论文主题是否在该期刊Scope范围内？（给出判断依据）
□ 研究方法是否符合该期刊偏好的方法论？
□ 数据集/样本规模是否达到该期刊的期望水平？
□ 引用的文献是否有充分比例来自该期刊或同类期刊？

格式适配：
□ 字数是否符合要求（含/不含参考文献）？
□ 图表数量是否在限制内？
□ 参考文献格式是否为该期刊要求的格式？
□ 摘要字数是否符合要求？
□ 关键词数量是否符合要求（通常3-8个）？
□ 是否需要额外附件（补充材料/数据/代码）？

作者信息：
□ 是否需要Cover Letter？（给出该期刊的Cover Letter要求）
□ 是否需要推荐审稿人？（若是，给出推荐依据）
□ 是否需要利益冲突声明？

[粘贴目标期刊的Author Guidelines URL或关键要求]

逐条标注 ✅/⚠️/❌，生成完整的投稿前核查报告
```
