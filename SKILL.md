## 1. 文献阅读与分析

### 精读单篇文献
```
请你仔细阅读这篇文章，给出文章的方法、创新点、实验结果、应用场景，以及这项工作是如何实现持续学习的
```

### 总结研究领域现状
```
总结一下当前[领域名称]的研究现状，有视频的给出视频链接，详细具体给出国内和国外的研究现状，都是用的什么技术实现的，有视频的给出视频链接
```

### 综述撰写（多篇文献）
```
仔细阅读上面所有论文，根据论文写综述，引用标注为[论文完整标题]（不是编号），主题关于[主题方向]
```

### 研究现状罗列（含图）
```
给我[研究方向]的国内外研究现状，把最新的论文罗列出来，包括：文章标题、方法简述、与本工作区别、发表期刊/会议、发表时间、期刊级别，并呈现关键图
```

## 2. 论文写作与生成

### 从材料生成完整论文
```
这两个材料都是我的论文，其中[文件名]是最新且最正确的材料，另外一个包含图片，请结合两个材料生成我的最新论文，你是[期刊]编委，数学公式请生成可编辑的LaTeX版本，生成最终的论文word格式
```

### 修改核心创新点表达（非并列结构）
```
我论文最核心的创新是[核心描述]，其它的作为辅助。你现在写法像并列结构，需要修改这种结构，让核心创新点突出，按照我说的来写，生成修改后的完整中文论文word文档
```

### 生成引言（含参考文献）
```
生成完整的引言，以及引言所包含的所有参考文献（标题，发表时间，所属期刊或会议）
```

### 生成相关工作（含参考文献）
```
生成完整的相关工作，相关工作部分需要引用5篇[目标期刊]的[年份范围]的研究，其它研究需要最新的一区二区和顶会，给出所有引用参考文献（标题、发表时间、期刊/会议）
```

### 引言/相关工作写法规范
```
引言和相关工作要改成：谁提出了什么方法，做了什么工作，有什么局限性，我们针对这些局限性做了什么。一定要客观，禁止批评别人的方法，禁止踩一贬一
```

### SCI一区论文写作指导（从数据/毕设起步）
```
这个是我的[毕业论文/实验数据]，请你仔细阅读所有材料，我现在要在现有资料基础上写一篇SCI一区论文，给我一个详细完整的论文写作指导和全文框架，以及生成一篇论文，所有内容生成完整word在桌面上
```

## 3. 论文润色与修改

### 全文语法/标点错误检查
```
检查一下这个论文里有没有明显的错误，语法，标点符号，表达等，这个将要发表，帮我检查有没有错误
```

### 段落学术化改写
```
从学术论文的角度修改这段话
```

### 顶刊标准全文纠错
```
你是顶刊期刊编委和审稿人和学术专家，技术大牛，帮我纠正我的论文错误，全文纠正，包括方法，实验，引言，摘要，结论
```

### IJCAI审稿人视角修改（含overleaf对照）
```
你是IJCAI顶会审稿人，并且还是主编，告诉我这篇文章在逻辑、语句、原理方法、实验结果上是否存在致命的会导致拒稿的问题？请一一罗列并告诉我如何修改，给出具体替换语句，生成中英对照版和overleaf格式版本，详细标注修改的地方
```

### 论文精简（保留原意，含overleaf）
```
精简一点（保留原意和原overleaf格式），去掉冗余部分重新生成中英对照版本，不要丢失参考文献。需要详细告诉我原来是什么，如何修改，在哪里修改
```

### 按TRO风格润色（方法/实验/相关工作）
```
这个是论文[部分名称]，帮我按照SCI一区、TRO的学术语言风格进行润色，要求逻辑缜密，如果原理不对的话请修改出来，生成修改后的完整[部分名称]
```

### IEEE TRO冗余精简（含latex修订）
```
你是IEEE TRO审稿人和编委，文章现在太啰嗦冗余了，保留核心简要的，删去冗余，告诉我具体修改哪里，用什么句子替换，不要省略，生成完整修改说明中文word
```

### 公式符号规范检查
```
根据我论文的方法部分，帮我看一下图里面的公式字母表述对不对，图里面有公式字母没有对应上论文方法部分的描述，详细告诉我如何修改。要求：变量斜体，函数/常量正体，字母下标不要用英文单词，变量简洁，生成完整修改意见word，公式需要latex格式
```

### 实验数据更新（含标红对比）
```
[描述新增实验数据变化]，帮我更新论文里所有表格数据，包括论文部分所有涉及到这个的描述，生成完整版论文，你修改的地方标红，防止乱码，我是wps打开的，生成完整中文版word
```

## 4. 论文审稿与投稿

### 论文总结与优缺点
```
总结全文，你是审稿人，给出论文总结、优缺点，以及明显的问题
```

### 指定期刊投稿意见
```
我要投稿的期刊是[期刊名]，给我关于论文的修改意见以及绘图意见，详细准确，分析我的论文结构有没有需要改进的地方
```

### Nature Communication 写作
```
这是我的研究内容，我计划投稿nature communication，给出投稿建议，给出论文标题，你是nature审稿人，根据我的研究内容帮我直接写一篇完整的nature communication
```

### IEEE TRO 找拒稿漏洞（含修改清单）
```
你是IEEE TRO审稿人，帮我找到会导致拒稿的致命漏洞，包括逻辑，理论，实验，以及全文，在不补实验的基础上，给我一个修改清单，包括修改哪里，如何修改，用什么描述语言替换，生成完整word
```

### IEEE TRO 审稿意见（中文，含latex）
```
你是IEEE TRO审稿人和编委，给出审稿意见和最终结果，检查致命问题，在不补实验基础上给详细修改说明。给我中文版修改意见，修订直接在latex上修改，给我可以复制进去替换的latex代码，精确告诉我修改哪一行，修改前后对比都要给出来，解释为什么这样修改，生成完整word。
要求：LaTeX代码块精确可搜索；排版清晰，修改前/后严格对齐
```

### IEEE TII 审稿人视角（全文审阅）
```
你是IEEE TII顶刊SCI一区审稿人，并且还是主编，告诉我这篇文章是否存在致命的会导致拒稿的问题，请一一在原文中标注出来，详细告诉我如何修改，给出具体修改替换语句，生成标注后的pdf在桌面上
```

### IJCAI 发表前全文校对（含latex）
```
帮我对这篇即将发表的文章进行全文检查，检查是否有图片字体错误、单词拼写错误、参考文献错误、表格错误或其他明显错误，帮我进行校对勘误，生成详细具体修订word，修订直接在latex上修改，给我可以复制进去替换的latex代码，生成word在桌面上
修订报告格式参考：[参考word路径]
```

### 小修后对照检查（TII）
```
你是IEEE TII审稿人，检查这篇论文是否按照小修意见进行了修改，检查图片字体、单词拼写、参考文献、表格是否有明显错误，帮我校对勘误，生成详细具体修订word
```

### 转博/IJCAI对比审稿
```
我上传了一篇我自己的论文和[年份]IJCAI接收的论文，你作为审稿人和主编，给出我的论文的最终决定意见
```

## 5. 论文翻译

### 全文中译英（学术化，Nature标准）
```
把论文全文翻译为英文，不要丢失参考文献标号和公式，你是nature主编，需要百分百确保翻译后的英文意思和我原来的中文一模一样，不要改变我的意思，需要非常准确及学术化，生成完整word在桌面
```

### 部分章节中译英
```
把论文的[摘要/引言/相关工作/公式部分]翻译为英文，不要丢失参考文献标号和公式，并为公式编号，生成完整word在桌面，[其它部分]保持原来的中文不变
```

### 翻译准确性校对（标蓝标红）
```
对比一下翻译的是否准确，不准确的地方请帮我修改，修改前标蓝，修改后标红，并中文说明修改原因，生成修改后的完整word在桌面上
```

### 实验部分翻译校对
```
对比一下实验部分的翻译是否准确，不准确的地方请帮我修改，修改前标蓝，修改后标红，中文说明修改原因，生成修改后的完整word和详细修改说明报告（只标红要修改的句子，不要大段大段标红），只对比实验部分
```

### 审稿意见英译中
```
把这个审稿意见翻译为中文版本的word在[目标路径]下，帮我排版好，以及给我如何修改论文的建议
```

## Common Mistakes

- **提示词太短**：审稿类提示词需要明确说明"不补实验的基础上"、"生成word在桌面上"等关键约束，否则AI会自由发挥
- **忘记指定输出格式**：每次都要说清楚生成word/latex/pdf，以及生成到哪个路径
- **润色vs写作混用**：润色是在原文基础上改，写作是重新生成，给AI的角色定位要区分清楚
- **latex修订要精确**：要加上"精确告诉我修改哪一行，修改前和修改后的对比"，否则找不到修改位置
