# 学术绘图与图表提示词

本模块专门服务论文图表：方法流程图、实验结果图、图题图注、表格转图、SCI 一区/Nature 风格图示。输出提示词时默认只生成绘图提示词，不夹杂无关说明。

---

## 1. 方法流程图提示词

```text
这是我的论文/代码/方法描述，我要投稿 SCI 一区。请根据我的方法生成一个用于 [绘图工具名称] 的完整 AI 绘图提示词，用于绘制方法流程图。

要求：
1. 只输出一个完整提示词，不要输出解释；
2. 图要学术、美观、Nature/AI 顶会风格；
3. 清晰体现核心算法流程，而不是普通工程模块堆叠；
4. 包括配色、布局、图标、元素、箭头、模块层级、输入输出、关键公式位置；
5. 使用专业视觉语言：多尺度、闭环、跨域泛化、环境侧感知、视觉-触觉融合、策略更新等；
6. 不要生成与论文无关的元素；
7. 如果涉及机器人，画出传感器、机器人本体、目标物体、反馈闭环和任务结果。
```

## 2. 实验结果图提示词

```text
这是我的论文和实验描述。我需要重制实验 [编号/名称] 的结果图，用于 SCI 一区投稿。

实验目的：[实验目的]
实验变量：[变量，例如目标位置、相机位置、机器人位置、物体类别]
关键结果：[成功率/误差/耗时/泛化能力等]

请生成一个用于 [绘图工具名称] 的完整提示词。

要求：
1. 只输出提示词；
2. 图要体现实验变量变化和结果稳定性；
3. 如果涉及空间位置变化，使用网格、坐标轴、轨迹、姿态变化示意；
4. 如果涉及多物体/多平台，使用分组布局或矩阵布局；
5. 颜色学术克制，突出重点数据；
6. 包含标题、子图编号、图例、坐标轴、误差条/置信区间/成功失败标记；
7. 适合论文正文直接引用。
```

## 3. 表格转学术图

```text
这是我的论文表格 [表编号/表名]。请判断它是否适合继续保留为表格，还是应改成更直观的学术图。

请输出：
1. 推荐图型：柱状图、折线图、热力图、雷达图、散点图、矩阵图、流程对比图或混合图；
2. 为什么这种图比原表更适合；
3. 图中应该保留哪些数据，删除哪些冗余信息；
4. 子图布局；
5. 坐标轴、图例、颜色、字体、标注规范；
6. 一条可直接用于 [绘图工具名称] 的完整绘图提示词。
```

## 4. 图题、图注和正文引用

```text
请根据这张图和论文上下文，生成：
1. 学术化图题；
2. 完整图注；
3. 论文正文中引用该图的一段文字。

要求：
1. 图题简洁，突出该图验证的核心问题；
2. 图注说明每个子图含义、实验设置、变量、结果和结论；
3. 正文引用不能只是“如图所示”，要说明该图支撑了什么论点；
4. 语言符合 [目标期刊/会议] 风格。
```

## 5. 图表质量审计

```text
请以 SCI 一区论文图表标准审查这张图/这个表。

检查：
1. 图是否真正支撑论文主张；
2. 是否存在字体过小、颜色混乱、图例不清、坐标轴缺失、单位缺失；
3. 子图之间是否逻辑连续；
4. 是否存在重复展示或无效装饰；
5. 是否需要合并、拆分或重排；
6. 图题和图注是否能独立解释图中信息。

输出：问题清单 + 修改建议 + 可直接替换的图注。
```
