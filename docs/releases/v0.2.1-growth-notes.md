# v0.2.1 Growth Notes

Date: 2026-07-06

## Goal

Improve GitHub visitor conversion: make the value clear in the first screen, give a fast demo, and make contribution requests easier.

## Changes

- Added badges to the v0.2 and English README pages.
- Added a "Who It Is For" section to clarify the target user.
- Added a 60-second demo to make the main workflow visible without reading the whole skill.
- Added a "Star This Repo If" section to make the call to action explicit.
- Added issue templates for examples, bugs, and feature requests.
- Added a full submission package example.
- Added a demo video script for social posts.

## Growth Hypothesis

Stars increase when visitors can answer four questions quickly:

1. Is this for me?
2. What problem does it solve?
3. Can I try it in one minute?
4. Is there proof that it handles real workflows?

v0.2.1 addresses these four questions without changing the main skill behavior.
