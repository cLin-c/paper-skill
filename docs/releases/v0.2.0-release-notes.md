# v0.2.0 Release Notes

Date: 2026-07-06

## Goal

Make paper-skill easier to understand, demo, and share without replacing the existing README or changing the main SKILL.md.

## What Changed

- Added [README.v0.2.md](README.v0.2.md), a sharper landing page focused on the core promise: Chinese draft to English submission package.
- Added an [example gallery](../examples/README.md) with three copyable demonstrations.
- Added concrete examples for:
  - Chinese abstract to academic English.
  - Reviewer response drafting.
  - Citation integrity and over-claim audit.
- Added a minimal pointer in the original README so visitors can discover the v0.2 materials.

## Positioning Update

The new positioning is:

> Chinese-first academic paper skill for SCI / Nature / IEEE submissions.

The intended first-viewport message is:

> Turn Chinese research materials into a submission-ready English package with writing, reviewer response, citation integrity, and quality gates.

## Why This Helps Growth

- Visitors can understand the project in 30 seconds.
- Users can copy a first prompt immediately.
- The project now has reusable examples, which are easier to share on GitHub, Zhihu, Xiaohongshu, Bilibili, and lab groups.
- The value proposition is narrower and more memorable than "22 scenarios".
