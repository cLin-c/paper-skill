# v0.4.0 Toolchain and Examples Notes

Date: 2026-07-06

## Goal

Address three maturity gaps: executable checks, visible before/after examples, and clearer international positioning.

## Changes

- Added `tools/paper_skill_gate.py`, a zero-dependency stage-gate checker for submission package JSON files.
- Added `tests/test_paper_skill_gate.py` and GitHub Actions workflow `toolchain.yml`.
- Added `examples/submission-package-check.json` as a runnable sample package.
- Added before/after examples for Chinese abstract rewriting, reviewer response, and citation integrity audit.
- Added `docs/toolchain.md` to explain the executable workflow.
- Updated README and README_EN with toolchain links and non-native English researcher positioning.
- Updated `VERSION` to `0.4.0-preview`.

## Result

paper-skill is no longer only a prompt library. It now has a small but concrete validation path that users can run locally or in CI.
